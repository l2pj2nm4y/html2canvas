Testing modern CSS color functions: oklch() and oklab()

What this tests: Support for OKLCH and OKLAB color spaces, which are perceptually uniform color spaces.
OKLCH: Lightness, Chroma, Hue (cylindrical coordinates)
OKLAB: Lightness, a (green-red), b (blue-yellow) (rectangular coordinates)
